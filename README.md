To run this project, install Docker Desktop, clone this repo, and then run the following command from the root folder:

docker compose up --build

Then go to localhost:4000 and you can draw in the canvas.

Be warned, it will take a while to run the docker command the first time you do this. 
Everything is fine, pytorch is just a big library.

Drawings the model can predit:
Airplane
Bicycle
Book
Car
Chair
Clock
House
Telephone
Tree
Umbrella

For best results, draw one image at a time. Draw the image in the center of the canvas with even padding around.
